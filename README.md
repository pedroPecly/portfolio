# Portfolio React (Next.js)

Portfolio responsivo em Next.js (App Router) com Tailwind CSS e layout pronto para personalizacao.

## Como rodar

1. Instale dependencias: `npm install`
2. Inicie o servidor: `npm run dev`
3. Abra o app no navegador.

## Onde editar

- `src/app/page.tsx`: conteudo e secoes.
- `src/app/globals.css`: cores, tipografia e animacoes.
- `src/app/layout.tsx`: fontes e metadados.

## Personalizacao rapida

- Troque "Seu Nome", contatos e textos de exemplo.
- Substitua cards de projetos, experiencia e habilidades.
- Ajuste cores em `:root` no `globals.css`.

## Observacoes

- O formulario de contato e demonstrativo; conecte a um backend ou servico quando desejar.
